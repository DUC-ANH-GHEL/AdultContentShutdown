Privacy Policy for Adult Content Shutdown Guard

Effective date: June 18, 2026

Adult Content Shutdown Guard is a supervised browsing protection extension for Microsoft Edge. The extension works together with a locally installed Windows companion service and Microsoft Edge managed policy settings.

1. Purpose

The extension is designed to help users or administrators apply a configured protection action when restricted websites are visited. Its single purpose is supervised browsing protection.

2. Data the extension may process

The extension may process browser tab and navigation information, including visited URLs, in order to determine whether a webpage matches the configured restricted-site protection rules.

The extension may also read managed policy configuration from Microsoft Edge, including the local companion service URL and authorization settings.

3. How data is used

The data is used only to evaluate whether the current webpage matches the configured protection rules and to communicate the protection event to the locally installed Windows companion service.

4. Local communication

The extension communicates with the local Windows companion service at a configured local address such as http://127.0.0.1:8765/. This communication happens on the user's device.

5. Data sharing

The extension does not sell user data.

The extension does not transfer user data to third parties.

The extension does not use user data for advertising, profiling, creditworthiness, or lending purposes.

The extension does not transmit browsing data to a third-party server.

6. Remote code

The extension does not execute remote code. All extension code is included in the submitted extension package. Configuration is provided through Microsoft Edge managed storage.

7. Data storage

The extension does not store personal browsing history in extension storage. Configuration is read from Microsoft Edge managed policy.

8. User control

The extension is intended to be installed and configured by the user or administrator. Users or administrators can remove the extension, remove the local companion service, or update the managed policy configuration according to their device management setup.
